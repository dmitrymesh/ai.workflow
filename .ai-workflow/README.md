# .ai-workflow

This folder contains the local AI task protocol for this repository.

## Quick start

```bash
python .ai-workflow/scripts/ai_task.py create "Example task"
python .ai-workflow/scripts/ai_task.py move AI-001 ready
python .ai-workflow/scripts/ai_task.py board
```

## Folder model

Task status is determined by directory:

```text
tasks/draft/
tasks/ready/
tasks/in_progress/
tasks/ready_for_review/
tasks/changes_requested/
tasks/ready_for_human/
tasks/done/
tasks/rejected/
```

Each task is a folder containing:

```text
metadata.yaml
task.md
report.md
review.md
decision.yaml
validation.md
```

## Board

`board.md` is generated by:

```bash
python .ai-workflow/scripts/ai_task.py board
```

Do not edit it manually.

## Validation

Run:

```bash
python .ai-workflow/scripts/ai_task.py validate
```
